# Quizz It

Aplicação web de quiz desenvolvida com **Node.js**, **Express** e **JavaScript**.

O usuário escolhe uma categoria, responde perguntas geradas dinamicamente a partir da API **Open Trivia DB** e, ao final, visualiza o resultado com a quantidade de acertos.

Projeto simples para aprender a consumir uma API, manipulação de dados no backend e renderização dinâmica com EJS. 

---

## Tecnologias

- Node.js  
- Express  
- JavaScript  
- EJS  
- CSS  

---

## Como executar

1. Instale as dependências:
```bash
npm install
```

2. Crie o arquivo .env:
```env
SESSION_SECRET=sua_chave_secreta (insira sua chave)
```

3. No terminal rode:
```bash
node server.js  
```

4. Acesse no navegador, colocando a seguinte URL:
```md
http://localhost:3000
```
